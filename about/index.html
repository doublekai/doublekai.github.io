核心架构：

技术理解

Agent vs 传统 LLM

模式 流程 特点
传统 LLM 提问 → 回答 只能基于训练数据
Agent 提问 → 思考 → 调用工具 → 获取结果 → 回答 可连接真实数据

SSE vs WebSocket

特性 SSE WebSocket
方向 单向（服务端→客户端） 双向
复杂度 简单，基于 HTTP 较复杂，独立协议
适用场景 流式输出、通知推送 聊天、游戏
